# Todo CRUD with Axios

Aplicacion de gestion de todos. Agregar, Editar, Eliminar todos y Guardar registros en un archivo en nuestro directorio.

## Descripción

Aplicacion simple de todos, utilizando server-json para simular backend, se utiliza la libreria axios para poder aplicar el get,post,put y delete.

## Instalación

Sigue estos pasos para instalar la aplicacion en tu sistema:

1. Clona este repositorio en tu máquina local
2. Navega a la carpeta e instala las depencencias - "npm install"
3. Esta aplicacion necesita de json-server del archivo db.json en el puerto 3001 para levantar la REST API - levanta el json server con el comando "npm run start-server".
